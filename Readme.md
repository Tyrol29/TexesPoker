# 快速开始

## 运行游戏

```bash
cd texas_holdem
python main.py
```

## 菜单选项

1. **开始新游戏** - 交互式对战模式
2. **运行测试** - 自动测试AI对战
3. **游戏规则** - 查看德州扑克规则
4. **退出** - 退出程序

## 游戏设置

1. 选择玩家人数 (2-8)
2. 选择AI数量 (1到玩家数-1)
3. 输入你的名称
4. 开始对战！

## 常用命令

| 输入 | 行动 |
|------|------|
| `f` 或 `fold` | 弃牌 |
| `c` 或 `call` | 跟注 |
| `k` 或 `check` | 过牌 |
| `r [金额]` 或 `raise [金额]` | 加注 |
| `a` 或 `allin` | 全押 |

## 提示

- 电脑名称显示其打法风格（如`电脑1号[紧凶]`）
- 每100手牌后会输出统计报告
- 第一个电脑被淘汰后，盲注会翻倍
- 你的底牌会显示，AI的底牌会隐藏

## 示例

```
========================================
底池总额: 150 筹码
========================================

你的回合
```
